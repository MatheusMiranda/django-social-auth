Login with GitHub

Django application that allow users to add Tags to their repositories on Github, helping them to classificate and organize more easily their projects.

You can click on the red button to ask questions about this application, to send suggestions of improvements and to relate bugs.

About this technology How to collaborate Contact and License
  • Simple application that implements integration between Django and Github, using the python lib social-auth-app-django and allow users to add tags on their repositories, classifing and filtering them by categories.
  • This project is a test app, that serve as a boilerplate for projects that use Django Social Authentication.

How to use:

  • Click on Login with GitHub and permit authentication on your Github Oauth App
  • Click on 'Update my Repositories' to fetch your repositories data from GitHub.
  • Create 'tags' or add them to your repositories by clicking on button 'Edit Repository'
  • The 'search' field can be used to filter repositories according to its 'tags'

All the code is located at the: Django Social Github repository
